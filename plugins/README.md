These are plugins for agent runtime harnesses like claude-code

Primarily we only support claude-code, but there may be a plan later to also support opencode

Agent harnesses like qwen code are compatible with claude code plugins

For a wide array of info on agent harnesses, see (non-exhaustive list):

- https://github.com/ai-boost/awesome-harness-engineering
- https://github.com/AutoJunjie/awesome-agent-harness
